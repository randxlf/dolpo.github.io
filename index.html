Project 2


Course Data Manager: CSV Analysis and Reporting
Tool for Coursera Users


This project does a specific job. It takes two CSV files as input. The first file
is a report from Coursera about users, and the second file is a master list
containing people's names, emails, and where they're from.

The project first checks if these files are the right ones and if they have any
missing information. It also makes sure the information in the rows is correct.
Once the data is checked and imported, the project tells the user if there are
any duplicate accounts.

Then, the user can use filters and sorting to create a new set of information
and save it to a new CSV file. The new information will have an extra column
showing how much time has passed between when the person signed up and
when they finished their last course. This time is shown in months and days.

Project 3


Registration System Module


The Registration System Module is an innovative web-based module designed to
facilitate seamless enrollment in specialized subjects, such as Java programming and
more, within the Department of Information and Communications Technology (DICT).
This system offers users the flexibility to choose between free and premium courses
based on their preferences and needs.

The Registration System Module is a cutting-edge solution that streamlines the course
enrollment process within the DICT. By offering a wide array of subjects, allowing users
to choose between free and premium options, and providing administrators with robust
tools to manage enrollments effectively, this system revolutionizes the way courses are
accessed and managed.

This project represents a pivotal step toward digital transformation within educational
institutions and reflects a commitment to modernizing administrative processes while
prioritizing user convenience and engagement.
